Компонент `_Предпросмотр файла_` позволяет просматривать содержимое файла карточки без его открытия во внешнем приложении и скачивания на устройство.

== Открытие предпросмотра файла

Чтобы открыть предварительный просмотр содержимого файла, выберите его в списке файлов карточки.

Для компонента `_Предпросмотр файла_`, встроенного в разметку, также доступна возможность выбора просматриваемого файла из списка в самом компоненте. При этом файл будет выделен в списке файлов карточки.

.Выбор файла для предпросмотра из списка
image::file-preview-files.png[Выбор файла для предпросмотра из списка]

include::layouts:page$ctrl/special/filePreview.adoc[tags=previewFormats]

Если формат файла не поддерживается, будет предоставлена ссылка для его сохранения на компьютер пользователя.

.Предпросмотр неподдерживаемого типа файла
image::file-preview-unavailable.png[Предпросмотр неподдерживаемого типа файла]

При предпросмотре текстовых файлов (с расширением `.txt`) отображаются только первые 100 000 символов. Если файл содержит большее количество символов, в предпросмотр будет добавлена строка `Полный предпросмотр данный файла недоступен по причине большого размера`. Чтобы посмотреть файл полностью, сохраните его на устройство и откройте в стандартном средстве просмотра.

== Комментирование

Если предпросмотр открывается в отдельном окне, область комментариев будет доступна сразу. Чтобы прокомментировать файл в компоненте `_Предпросмотр файла_`, встроенном в разметку, нажмите image:buttons/comments-contour.png[Комментарии], если комментариев нет и image:buttons/comments-contour-blue-dot.png[Иконка комментарии с синей точкой] если комментарии есть. Закрыть блок комментирования можно повторным нажатием кнопки или клавишей kbd:[Escape].

.Комментирование файла
image::file-preview-comments.png[Комментирование файла]

[NOTE]
====

== Поиск по документу

Нажмите кнопку image:buttons/search-black.png[Лупа] на панели инструментов и введите искомый текст в поле поиска.

.Поиск по документу в окне предпросмотра
image::file-preview-search.png[Поиск по документу в окне предпросмотра]

Поиск возможен только при просмотре текстового документа.

== Изменение масштаба

Для изменения масштаба документа используйте кнопки *-* / *+* или список справа от кнопок.

Для компонента `_Предпросмотр файла_`, встроенного в разметку, также доступна возможность расширения области предпросмотра. Чтобы расширить область предпросмотра, наведите курсор на границу предпросмотра и данных карточки так, чтобы изменилась форма курсора. Потяните разделитель вправо или влево, чтобы изменить ширину области предпросмотра.

.Изменение ширины области предпросмотра
image::file-preview-change-width.png[Изменение ширины области предпросмотра]

== Другие функции предпросмотра

Компонент предпросмотра предоставляет также другие функции: печать, сохранение документа и др.

Нажмите кнопку image:buttons/bracket-arrows.png[Двойная стрелка-галочка вправо], чтобы открыть полный список доступных команд.
