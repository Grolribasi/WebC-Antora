= Тулбар папки

Панель управления нового грида. Свойства панели управления регулируют операции, доступные для представления папки.

Элемент управления является контейнером, который может быть расширен добавлением новых команд. Панель управления можно расширять при помощи добавления ЭУ xref:layouts:ctrl/standard/layoutIconButton.adoc["Кнопка-иконка"]. Каждая новая кнопка соответствует команде.

Чтобы добавленная кнопка работала как команда, необходимо настроить её свойства, включая CSS класс, обработчик события, отображаемое название функции (название ЭУ). Если для кнопки не задан CSS класс, по умолчанию кнопка будет иметь следующий вид: image:buttons/button-icon.png[Нажатый курсор]. Если не задано название кнопки, будет отображаться название ЭУ по умолчанию.

Элемент управления "Новая команда" image:buttons/new-command.png[] нужно добавить сюда. См. подробнее xref:ctrl/folderComponents/new-command.adoc[].

=== Внешний вид

Свойства категории "Внешний вид" могут быть активированы или отключены отдельно для каждого типа устройств: Десктоп, Планшет, Телефон. Например, чтобы отключить свойство, для телефона, нужно снять флаг в строке "Телефон".

[#display]
==== Изменить отображение

Позволяет или запрещает пользователям переключаться между режимами: Грид или Лента.

[#settings]
==== Настройки

Регулирует доступность окна пользовательских настроек с панели инструментов.

[#refresh]
==== Обновить

Позволяет отключить кнопку обновления данных в представлении.

[#read]
==== Прочитать все

Позволяет отключить кнопку обновления данных в представлении.

[#size]
==== Размер папки

Включает или отключает отображение количества карточек в папке.

[#position]
==== Расположение дополнительных элементов

Определяет, где по отношению к стандартным кнопкам панели усправления будут располагаться дополнительные элементы управления (добавленные кнопки): *_Слева_* или *_Справа_*.

[#reset]
==== Сбросить настройки

Когда свойство активно, пользователю будет доступна кнопка "Сбросить пользовательские настройки". Когда свойство неактивно, сбросить настройки на стандартные нельзя.

[#export]
==== Экспорт

Разрешает или запрещает возможность экспортировать представление в Excel.
